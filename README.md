# Besnik Landing Page (HTML & CSS)

This project is a frontend recreation of the Besnik Figma design using only HTML and CSS, built completely from scratch. It focuses on strengthening core frontend skills like layout design, responsiveness, and clean styling without using any frameworks or libraries.

## 🚀 Features
- Fully responsive design
- Built with pure HTML5 & CSS3
- Flexbox layout usage
- Clean and semantic code structure

## 🎯 Purpose
To practice and improve fundamental frontend development skills by converting a Figma UI design into a real-world web page.

## 📁 Tech Stack
- HTML5
- CSS3
